# 🏛️ ITSE License Expiration Dashboard - Chiclayo, Peru

## 📋 Overview
Interactive Power BI dashboard analyzing the expiration status of Technical Safety Inspection (ITSE) licenses issued in Chiclayo, Peru, covering the period 2024-2026.

## 🎯 Objectives
- Monitor licenses that are expired, about to expire, or currently valid
- Identify commercial sectors with the highest number of expiring licenses
- Support municipal decision-making through data visualization

## 📊 Dashboard Features
- **KPI Cards:** Total expired, expiring within 30 days, and active licenses
- **Bar Chart:** License expirations grouped by month
- **Top 10 Chart:** Commercial sectors with the most expiring licenses
- **Traffic Light Table:** Detailed view of each business with color-coded status (🔴 Expired / 🟡 Expiring soon / 🟢 Active)

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query

## 📁 Data Source
Official ITSE registration dataset from the Municipality of Chiclayo (2024-2026)
